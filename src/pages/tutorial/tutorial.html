Agrofloresta!

Esta é uma plataforma de construção colaborativa de conhecimento sobre Agroflorestas.

Seja bem vindo!

Objetivo

A idéia é criar-mos uma base de dados e um canal de comunicação e difusão dos conhecimentos sobre agrofloresta.

Caracteristicas

  • Conteúdo criado e mantido de forma colaborativa.
  • Qualquer pessoa pode criar ou alterar qualquer conteúdo.
  • O Código fonte é aberto (open source).
  • Conteúdo disponível offline.
  • Sem banners e publicidade.
  • Disponível nas plataformas web, android e ios.

Como colaborar

  • Faça sua propria agrofloresta!
  • Ajude a criar e manter o conteúdo na plataforma.
  • Colabore no código ou design.
  • Informe qualquer problema.
  • Sugira uma melhoria.
  • Faça uma doação!
